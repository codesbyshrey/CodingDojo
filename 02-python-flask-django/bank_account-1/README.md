## Bank Account (Procedural Python)

This folder contains a single `bank_account.py` file that implements **looping and arithmetic exercises** themed as bank account operations.

### Overview

- **Goal**: Practice Python basics such as variables, arithmetic, and printing output, framed as bank‑style computations.
- **Core idea**: Simulate account operations purely in the console without classes or a database.
- **Tech stack**: Python 3 (no external dependencies).

### How to Run Locally

1. Ensure Python 3 is installed.
2. From this directory run:
   - `python bank_account.py`
3. Watch the console output:
   - The script prints results for the different exercises it contains.

### Learning Focus

- Strengthens:
  - Comfort with basic Python syntax.
  - Using variables to track balances.
  - Printing descriptive results to the console.

### Portfolio Notes

- This is an early exercise and not a full application UI.
- It still demonstrates growth from simple math problems toward domain‑inspired logic (banking).

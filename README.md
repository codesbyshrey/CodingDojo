# CodingDojo Full-Time Bootcamp

January 23, 2023 - April 28, 2023

This repository is organized by bootcamp learning block, with source projects preserved and generated artifacts removed.

## Structure

- `00-pre-bootcamp`: early setup and warmup exercises.
- `01-web-fundamentals`: HTML, CSS, JavaScript, DOM, and static frontend assignments.
- `02-python-flask-django`: Python fundamentals, Flask, MySQL, full-stack Python projects, and data science notes.
- `03-mern`: JavaScript, Express, React, MongoDB, and MERN-stack projects.
- `04-java-spring`: Java fundamentals, OOP, Spring, Maven, and STS workspace projects.
- `05-algorithms`: algorithm and data-structure practice separated from the project blocks where practical.
- `99-archive-review`: material kept for later review that does not clearly belong to a main bootcamp block.

## Duplicate Naming

- `-1`: primary or more complete version when two related versions should both be kept.
- `-2`: secondary version with meaningful differences.
- `-copy`: duplicate-looking project with source differences or unclear provenance, preserved instead of deleted.

## Cleanup Notes

Generated folders and files such as `target`, `node_modules`, `__pycache__`, `__MACOSX`, `.metadata`, and `.DS_Store` were removed. Exact duplicate Spring exports were collapsed into the canonical Java Spring workspace copies, with assignment READMEs moved into the matching project folders.
